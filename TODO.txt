Название: LaserTank.
Предназначение: Управление танками с телефона через bluetooth
ЦА: гики, дети

Набор функций, выполняемых приложением:
	1. Авторизация в приложении.
	2. Поиск доступных роботов, подключение к роботу по Bluetooth.
	3. Управление роботом. 
	4. Авторизация в соцсети и приглашение друзей из нее.
	5. Заказ обновления или нового робота в интернет магазине. 
	6. Cбор и и показ статистики по играм.

Список используемых библиотек и сторонних технологий:
	1. VK&Facebook Android SDK
	2. Volley Http Client https://github.com/mcxiaoke/android-volley
	3. Moshi для парсинга JSON https://github.com/square/moshi
	4. SugarORM http://satyan.github.io/sugar/index.html
	5. https://github.com/mikepenz/MaterialDrawer

(1) Сплеш Экран (Алла)
		Лого
		Копирайты

(2) Экран регистрации (Алла)
		Логин
		Пароль
		Войти через соцсети

(3) Экран выбора робота (Гарри)
		Поиск доступных роботов
		Коннект
		Ввод пароля
		Плашки ошибок (пароль не тот, роботов нет)
	
(4) Экран Настройки: (Гарри)
		Имя робота
		В бой! -> (5)
		Смена пароля
		Использование гироскопа для управления
		Запомнить пароль
		Похвастаться -> (6)
		Прокачать в магазине -> (7)

(5) Консоль управления: (Алла)
		Стрелочки
		Кнопка выстрела
		Баллы за матч
		Заряд батареи
		Уведомления и подсказки (консоль с выводом сообщений от танка, как в миньоне было)

(M) Боковое меню: (Настя)
		В бой! -> (5)
		Настройки -> (4)
		Отключиться -> (3)
		Апгрейд танка -> (7)
		Пригласить друзей -> (6)
		Справка/отзыв -> web url
	
(6) Экран Пригласить друзей (Настя)
		Отправить письмо
		Сделать запись на стене

(7) Экран апгрейдов (Настя)
		Заказать апгрейд робота
		Заказать нового робота
